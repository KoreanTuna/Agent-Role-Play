0. 공통 규칙(전 화면)
   공통 UI 요소

상단: 제목 + (필요 시) 뒤로가기

기본 CTA: 하단 고정 버튼(가능한 화면에)

로딩: 스켈레톤/스피너

빈 상태: “왜 비었는지 + 다음 행동”을 반드시 제공

공통 예외 처리

네트워크 오류: “네트워크 연결을 확인해주세요” + 재시도

서버 오류: “일시적인 오류” + 재시도 + 문의

권한 문제(비공개): “상대가 정보를 공개하지 않았어요”

공개 범위(Privacy)

기본값: 친구만 / 일부 항목 비공개

공개 범위는 설정에서 변경하며, 화면에서 안내 배너로 표시 가능
예) “내 이직 정보는 현재 ‘친구만’에게 보여요”
