1. 서비스 개요

서비스명: CareerMap(커리어맵)
한줄 소개: 내 이력 + 동료 네트워크의 이직 흐름을 바탕으로 “다음 직장”과 “가능성이 높은 회사”를 예측·추천하는 커리어 네비게이션 앱

핵심 가치

사용자의 이력(경력/스킬/프로젝트)을 구조화해 개인 맞춤 직장 추천

함께 일했던 동료를 친구로 연결하고, 동료의 이직 데이터를 기반으로 내가 갈 수 있을 법한 회사(가능성 높은 회사) 예측

이직 준비에 필요한 “다음 한 걸음”을 명확히 제시(추천 이유/갭 분석/준비 체크리스트)

2. 문제 정의 & 해결 방식
   사용자가 겪는 문제

“내 이력으로 어디까지 갈 수 있지?”를 판단하기 어렵다.

주변 동료들의 이직 방향은 힌트가 되지만, 체계적으로 모아 보기가 어렵다.

추천 서비스는 많지만 내 네트워크 기반의 현실적인 추천은 부족하다.

커리어맵의 해결

이력 데이터를 표준화(직무, 도메인, 스킬, 임팩트 등) → 추천 엔진 입력값으로 사용

동료 네트워크 그래프(같이 일한 관계 + 동료의 이직 이벤트) → “경로 기반 예측”

추천 결과에 “왜 이 회사/직무인지”를 설명(설명가능 추천)

3. 타겟 사용자(초기)

1~10년차 직장인(개발/기획/디자인/마케팅 등)

이직을 고민 중이거나, 시장에서 내 포지션을 파악하고 싶은 사용자

“내 주변 사람들은 어디로 이동했는지”를 참고하고 싶은 사용자

4. 핵심 기능 정의(MVP 기준)
   A. 회원가입/로그인(필수)

이메일 회원가입/로그인

소셜 로그인(선택: Google/Apple 등) — MVP에서는 이메일만으로 시작 가능

비밀번호 재설정, 로그아웃, 탈퇴

B. 이력 입력(필수)

기본 프로필: 직무, 경력연차, 지역, 희망 조건(연봉/근무형태/업종 등)

경력 타임라인: 회사/기간/직책/팀/주요성과

스킬/키워드: 기술, 툴, 도메인, 자격증

프로젝트: 역할/성과지표/기여도(가능하면 정량)

C. “다음 직장 추천”(필수)

추천 결과 리스트(회사/직무/적합도 점수)

추천 이유(내 이력과 매칭된 포인트 3~5개)

갭 분석(부족한 스킬/경험) + “준비 액션” 제안

즐겨찾기/숨기기/관심 직무 태깅

D. 동료 친구 추가 & 네트워크(필수)

동료 추가 방식(초기 현실적인 옵션)

연락처 기반 초대(추천): 초대 링크/코드로 연결

또는 닉네임/코드 검색

동료 프로필(공개 범위 제한)

“같이 일했던 회사/기간”을 기반으로 연결 강도 계산(선택)

E. 동료 이직 흐름 기반 “가능성 높은 회사 예측”(필수)

동료의 이직 이벤트 입력/업데이트(사용자 본인/동료가 직접 업데이트)

예측 탭:

“내 네트워크에서 많이 이동한 회사 Top N”

“내 이력과 비슷한 동료가 간 회사”

“추천 경로” (예: A회사 → B회사로 이동한 동료 패턴)

예측 결과에도 이유/근거를 표시(동료 수, 경로, 유사도)

5. 화면/정보구조(IA)

온보딩

로그인/회원가입

기본 정보 입력(직무/연차/희망조건)

홈

오늘의 추천, 네트워크 하이라이트

이력(내 커리어)

경력/프로젝트/스킬 관리

추천

직장 추천 리스트 → 상세(이유/갭/액션)

네트워크

동료 목록, 초대/추가

동료 이동 타임라인(공개 범위 내)

예측

네트워크 기반 회사 예측, 근거 데이터

설정

계정/보안/공개범위/알림/탈퇴

6. 추천/예측 로직(기획 수준)
   (1) 직장 추천(콘텐츠 기반 + 규칙 혼합)

입력: 내 이력(직무/스킬/도메인/성과), 희망조건

방식:

스킬/직무 유사도 기반 매칭

경력연차/도메인 필터

사용자의 피드백(관심/숨김/지원함)으로 개인화

(2) 네트워크 기반 예측(그래프/유사도)

그래프: 사용자 ↔ 동료 ↔ 회사(이직 이벤트)

점수 예시:

동료 이동 빈도(최근 가중치)

“나와 유사한 동료”가 간 회사 가중치(유사도: 직무/스킬/경력)

연결 강도(같이 일한 기간/프로젝트) 가중치

중요한 UX 원칙: “예측”은 확정이 아니라 가능성으로 표현하고, 근거(숫자/경로)를 제공.

7. 데이터/권한/프라이버시 설계(핵심)

기본값: 비공개 중심

공개 범위 옵션:

내 프로필: 전체 공개 / 친구만 / 비공개

이직 이벤트: 회사명 공개/비공개(“대기업/스타트업” 레벨로만 공개도 가능)

동료 추가는 상호 승인(친구 요청/수락)

데이터 삭제/탈퇴 시: 개인정보 및 네트워크 노출 최소화(비식별 처리)

8. 알림(선택)

동료가 이직 업데이트 했을 때(친구 범위 내)

추천 결과 변동(새 추천 등장)

목표 스킬 학습 리마인드(갭 액션)

9. 운영/관리자(Admin) 기능(초기 최소)

유저/신고 관리(차단, 계정 정지)

콘텐츠/추천 품질 모니터링(추천 클릭률, 숨김률)

회사/직무/스킬 사전 관리(표준화)

10. 수익모델(선택)

프리미엄 구독:

예측 상세 근거 확장(경로/유사도 분석)

추천 무제한/필터 고급(연봉/스택/규모)

이력 진단 리포트 PDF

기업용(후속): 채용 브랜딩/추천 슬롯(사용자 동의 기반)

11. MVP 범위 & 일정 제안(기능 기준)

MVP(1차 출시)

이메일 회원가입/로그인

이력 입력(기본/경력/스킬)

직장 추천(기본 매칭 + 이유)

동료 추가(초대코드) + 동료 이직 이벤트 입력

네트워크 기반 회사 예측(Top N + 근거 요약)

공개범위/차단/탈퇴

2차(고도화)

소셜 로그인

추천 피드백 학습(개인화 강화)

네트워크 그래프 시각화

이력 자동 임포트(외부 연동은 정책/동의 설계 필수)

12. 성공 지표(KPI)

가입 → 이력 완성률(이력 입력 완료 비율)

동료 연결률(평균 친구 수, 초대 전환율)

추천 클릭률/저장률/숨김률

예측 탭 재방문율

4주 리텐션(주간 활성/재방문)
